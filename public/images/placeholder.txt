# Images Directory

This directory contains images for the AI Technology Journal website.

## Image Guidelines

- Use optimized images (WebP format preferred)
- Maximum width: 1200px for hero images
- Maximum width: 800px for article images
- Compress images to reduce file size
- Use descriptive filenames

## Placeholder Images
